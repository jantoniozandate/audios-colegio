## Design Context

### Users
Maestros y coordinación usan panel admin para grabar, corregir y publicar notas de voz para cada niño. Padres de alumnos usan pantalla pública para escuchar audio desde enlace directo o QR. Flujo debe ser rápido, claro y amable en contexto escolar.

### Brand Personality
Juguetón, cálido, infantil. Interfaz debe transmitir cuidado, cercanía y energía positiva sin verse caótica.

### Aesthetic Direction
Tema claro. Lenguaje visual editorial-lúdico con color suave, formas redondeadas, fondos atmosféricos y motion amable. Debe sentirse premium pero accesible para escuela y familias.

### Design Principles
1. Priorizar claridad operativa para maestros antes que decoración.
2. Usar calidez visual y microinteracciones para volver memorable experiencia.
3. Mantener superficies amplias, tipografía expresiva y jerarquía simple.
4. Diseñar QR y reproducción pública como flujo de un toque.
5. Respetar accesibilidad y `prefers-reduced-motion`.
